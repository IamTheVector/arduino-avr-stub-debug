# Packaged extension (VSIX)

This folder contains the **installable extension file** committed to Git so you can download it without building from source.

- **`arduino-avr-stub-debug-<version>.vsix`** (see `package.json` for `version`) — install this in Arduino IDE 2.x (or VS Code).

**How to install:** see the main [README](../README.md#install-the-extension-download-the-vsix).

Maintainers: regenerate after version bumps with `npm run package:release` from the repository root.
